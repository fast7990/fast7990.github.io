# 我的 Nuxt.js 应用

这是一个现代化的 Nuxt.js 应用程序，具有响应式设计和美观的用户界面。该应用目前包含三个页面：

- 首页 (/)
- 关于页面 (/about)
- 测试页面 (/test) - 包含用户信息表单和数据展示

## 项目结构

```
my-nuxt-app/
├── pages/
│   ├── index.vue (首页)
│   ├── about.vue (关于页面)
│   └── test.vue (测试页面 - 用户信息表单)
├── nuxt.config.ts (Nuxt 配置)
└── package.json (项目依赖)
```

## 开发

运行应用程序：

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 在浏览器中打开 http://localhost:3000

## 功能

- 页面间的基本路由
- 使用作用域 CSS 的现代化样式设计
- 响应式布局，适配各种设备屏幕尺寸
- 渐变背景和阴影效果
- 悬停动画和过渡效果
- 用户信息表单提交和数据展示（在测试页面）

### 页面详情

#### 首页 (/)
- 简洁的欢迎界面
- 导航链接到其他页面

#### 关于页面 (/about)
- 简单的关于页面内容
- 返回首页的导航链接

#### 测试页面 (/test)
- 用户信息表单（姓名、年龄、邮箱）
- 表单验证和提交功能
- 已提交信息的数据表格展示
- 时间戳记录功能
- 返回首页的导航链接

## 设计特点

所有页面都采用了统一的现代化设计风格，包括：
- 卡片式布局
- 渐变背景
- 圆角边框和阴影效果
- 平滑的悬停动画
- 响应式设计，适配移动设备
